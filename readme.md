## Installation 

### Requis
- Python >= 3.10
- Angular 13
- Postgresql
- 
(à compléter)

    git clone git@github.com:PremierLangage/PL2.git
    git checkout develop
    cd back-end/ 
    python3 -m venv env
    source env/bin/activate
    pip install -r requirements.txt


    cd ../front-end
    npm ci
